# 🐒 MONKO Games Collection

Two browser-based games featuring our favorite exiled jungle monkey on his path to zillions!

## 🎮 Games

### 1. Monko Jungle Run
An endless runner where Monko dodges obstacles and collects bananas while escaping the jungle.

**[Play Monko Jungle Run →](./jungle-run/index.html)**

### 2. Banana Clicker
Click to collect bananas and build your way to ZILLIONS! Upgrade your banana empire and watch the numbers go brrrr.

**[Play Banana Clicker →](./banana-clicker/index.html)**

## 🚀 How to Play

Just open the `index.html` file in any game folder with your browser. No installation needed!
